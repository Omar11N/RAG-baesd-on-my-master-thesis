Lightweight RAG System (Based on My Master's Thesis)

This repository contains a small, lightweight Retrieval-Augmented Generation (RAG) system that I developed based on my master's thesis. The system is designed to be local, free, and easy to set up for personal or research purposes.
Features

    1- Local Deployment: No external dependencies or cloud services required.
    2- Free and Open Source: Build and run without any subscription fees.
    3- Simple Setup: Minimal dependencies and straightforward configuration.
    4- Customizable: Easily adapt to your own datasets or retrieval mechanisms.
